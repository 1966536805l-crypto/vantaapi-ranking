# Project Goal

This project is an MVP learning website focused on English learning and C++ learning.

# Product Rules

- Build the MVP first.
- Do not add unnecessary complex features.
- Do not implement payment, live streaming, forum, or AI tutor in the first version.
- Do not implement online C++ code execution in the first version.
- Focus on course paths, lessons, quizzes, progress tracking, and wrong question review.

# Tech Stack

- Next.js
- TypeScript
- MySQL / MariaDB
- Prisma
- Tailwind CSS

# Code Rules

- Keep components small and reusable.
- Keep database models simple and extensible.
- Add clear comments for important logic.
- After each major change, explain how to run and test it.

# MVP Routes

- `/` home
- `/login` login
- `/register` register
- `/dashboard` user learning dashboard
- `/english` English learning home
- `/english/vocabulary` vocabulary
- `/english/grammar` grammar
- `/english/reading` reading
- `/english/quiz/[id]` English quiz
- `/cpp` C++ learning home
- `/cpp/basics` basic syntax
- `/cpp/oop` object-oriented programming
- `/cpp/stl` STL
- `/cpp/algorithms` basic algorithms
- `/cpp/quiz/[id]` C++ quiz
- `/admin` admin home
- `/admin/courses` course management
- `/admin/questions` question management
